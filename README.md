# Runestone IT Consulting Services

This repository contains a simple static company website for Runestone IT Consulting Services.

## Files

- `index.html` — main landing page
- `styles.css` — website styling
- `script.js` — simple interaction logic

## Run locally

1. Open `index.html` in your browser.
2. Or use a local web server, for example:

```bash
cd /workspaces/Stone
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
